🎮 Dynamic Steering System

Este projeto é uma aplicação desenvolvida para oferecer uma experiência completa ao jogar **Assetto Corsa utilizando mouse**, trazendo maior controle, precisão e competitividade para jogadores que não utilizam volante.

🚀 Visão Geral

O sistema transforma o uso do mouse em uma alternativa viável e altamente configurável para simulação de corrida, incorporando diversos recursos avançados normalmente encontrados em setups profissionais.

Além disso, o aplicativo oferece uma interface intuitiva que facilita a personalização de cada aspecto da pilotagem.

⚙️ Funcionalidades

O sistema inclui uma série de assistências e melhorias para otimizar o controle do veículo:

🎯 Force Feedback adaptado
  Simulação de retorno de força ajustada para entrada via mouse, proporcionando maior imersão e resposta dinâmica.

🛑 Sistema de Freios ABS
  Evita travamento das rodas durante frenagens intensas.

🏎️ Controle de Tração (Traction Control)
  Reduz a perda de aderência em acelerações, melhorando a estabilidade.

🔄 Auto-Blip
  Ajuste automático do acelerador em reduções de marcha.

⚙️ Auto-Clutch
  Gerenciamento automático da embreagem para trocas suaves.

🚫 Anti-Stall
  Previne o desligamento do motor em baixas rotações.

⬆️ No-Lift Shift
  Permite trocas de marcha sem tirar o pé do acelerador.

🚀 Launch Control
  Controle otimizado de largada para máxima aceleração.

🎚️ Scroll Gas
  Controle do acelerador utilizando o scroll do mouse.

🧩 Outras funcionalidades adicionais
  Diversos ajustes finos para adaptar o comportamento do carro ao estilo do jogador.

🎨 Personalização

Um dos principais diferenciais do sistema é a flexibilidade:

* Configuração individual de assistências
* Interface amigável para rápida alteração de parâmetros

🖥️ Interface

A aplicação conta com uma interface simples e eficiente, permitindo:

* Visualização em tempo real das configurações
* Alterações rápidas durante o uso
* Facilidade para novos usuários

🎯 Objetivo

O objetivo deste projeto é democratizar o acesso à simulação automotiva, permitindo que jogadores sem periféricos avançados (como volantes e pedais) possam competir e se divertir com alto nível de controle e realismo.
